# FFF - 发现收藏过的餐厅 (Favourite Food Finder)

基于百度地图收藏数据，智能发现当前位置1公里内被遗忘的收藏餐厅。

## 类型

Agent 型（单个 AI 专家）

## 核心功能

1. **位置获取**：通过浏览器地理定位或手动输入获取用户当前位置
2. **收藏数据管理**：解析百度地图导出的收藏数据，过滤餐厅类 POI
3. **空间计算**：基于 Haversine 公式精确计算距离，筛选 1 公里内的餐厅
4. **地图可视化**：生成交互式百度地图，标注收藏餐厅位置和距离

## 使用示例

- 帮我看看我现在附近有哪些我收藏过的餐厅
- 我在三里屯，帮我找1公里内我收藏过的餐厅
- 在地图上显示我收藏的所有餐厅，标出1公里范围内的

## 前置条件

使用本专家需要：
1. **百度地图 API Key (AK)**：在 https://lbsyun.baidu.com/ 注册并创建应用
2. **百度地图收藏数据**：从百度地图网页版导出收藏数据（我的 → 收藏 → 导出）
